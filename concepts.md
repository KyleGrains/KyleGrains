# Project Concepts

This page tracks recent directions I may turn into repositories or expand into
larger tools. Links are included when a public repository exists.

## Recent Directions

| Direction | What it is | Status |
| --- | --- | --- |
| [bytepet](https://github.com/KyleGrains/bytepet) | A terminal digital pet that ages, sleeps, plays, and changes through care. | Repo exists |
| `noveltech` | A reading companion that pairs public-domain fiction with technical books, so each session advances both tracks. | Local prototype |
| Desktop reading bridge | A local Codex bridge for Evince and Chrome, so the current PDF/page can become context without copy-paste. | Sketch |
| Dual-reading visualizer | A quiet HTML view showing novel progress, technical progress, and links between scenes and concepts. | Sketch |
| [configs](https://github.com/KyleGrains/configs) | Personal configuration files for Linux, editor, terminal, and Codex setup. | Repo exists |

## Principles

- Prefer tools that run locally and are easy to inspect.
- Keep interfaces simple before adding polish.
- Make reading, coding, and note-taking feel connected instead of separate.
- Build small prototypes first, then promote the useful ones into repos.
